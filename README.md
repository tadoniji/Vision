# Vision

Version serveur 1.22 (dernière stable)

## Guide d'utilisation

Pour utiliser **Vision**, voici les prérequis nécessaires :

- Node.js (18.x.x minimum) [Télécharger ici](https://nodejs.org/en/download)
- Un navigateur quel qu'il soit (Pour les navigateurs chromium, la version de noyau doit être supérieure à 100)
- Votre terminal (attention pas powershell) faites `Windows` + `R`, Tapes `cmd` dans la fenêtre qui vient de s'ouvrir et faites `Entrée`
- (Facultatif) Un bloqueur de pub efficace dans votre navigateur (Adguard, Ublock Origin/lite) pour un meilleur confort de navigation

### Étape 1 :

(si cela n'est pas déjà fait, téléchargez et installez node.js)

Ouvrez votre explorateur de fichiers, naviguez dans le dossier (attention, pensez à extraire l'archive si vous avez téléchargé l'appli en .zip) et copiez le chemin d'accès.
*Concrètement vous devez avoir un truc comme `C://Users/Me/Downloads/Vision`*

### Étape 2 : 

Dans le terminal tapez `cd [Chemin d'accès copié]` *Astuce : Clic droit pour coller du texte dans le terminal*

### Étape 3 : 

Si tout a bien été fait, vous pouvez taper la commande suivante `node server.js` là le programme vous donnera 2 adresses IP locales

La première : http://localhost:3000 et la seconde : http://192.168.x.x:3000

### Étape 4 : 

Ouvrez votre navigateur, utilisez la première adresse si votre fichier serveur tourne sur le même appareil que le navigateur, ou utilisez la seconde si vous voulez accéder à l'appli depuis un autre appareil.

## Avertissement concernant les versions :

Ce tutoriel est fonctionnel pour le téléchargement de la branche `Main` du dépôt, une appli "clé en main" existe dans la section release mais n'est pas encore fonctionnelle.
N'hésitez pas à contribuer si vous pouvez à l'amélioration de l'appli "clé en main".
